Sitios Responsivos con Media Queries

Las media queries (en español "consultas de medios") son útiles cuando deseas modificar tu página web o aplicación en función del tipo de dispositivo (como una impresora o una pantalla) o de características y parámetros específicos (como la resolución de la pantalla o el ancho del viewport del navegador)

Sintaxis
Las media queries consisten de un tipo de medio opcional y una o más expresiones de características de medios. Varias consultas se pueden combinar utilizando operadores lógicos. No distinguen entre mayúsculas y minúsculas. El resultado de la consulta es "verdadero" cuando el tipo de medio (si se especifica) coincide con el dispositivo en el que se está mostrando el documento y todas las expresiones en el media query son "verdaderas". En este caso, se aplica los estilos correspondientes, siguiendo las reglas usuales de cascada.

Media Types Los Media Types (tipos de medios) describen la categoría general de un dispositivo. Excepto cuando se utilizan los operadores lógicos not o only, el tipo de medio es opcional y será interpretada como all.

all
Apto para todos los dispositivos.
print
Destinado a material impreso y visualización de documentos en una pantalla en el modo de vista previa de impresión.

screen
Destinado principalmente a las pantallas.
speech
Destinado a sintetizadores de voz.

El HTML para SEO

Por su parte, HTLM se refiere en términos generales a un lenguaje informático utilizado para crear y ordenar el contenido que se encuentra dentro de una página web.
Este es utilizado para el desarrollo de los dominios. Son las siglas en inglés para HyperText Markup Language, o en español Lenguaje de Marcas de Hipertexto.
Las Marcas de Hipertexto son etiquetas o tags son increíblemente necesarias y útiles pues no solo ordenan el contenido, sino que además permiten que los buscadores puedan indexar de manera rápida las páginas web.
Es por esto, por lo que cuando se habla de HTML para SEO se refieren al uso de las tags para optimizar una página y que la misma pueda ser indexada.
